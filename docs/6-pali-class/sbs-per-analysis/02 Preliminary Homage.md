**Preliminary Homage**

|----------------------------------------------------------------------------------|------|--------------|---------------------------------------------------------------------------------------|
| handa                                                                            | ind  |              | well then; now                                                                        |
| mayaṃ                                                                            | pron | 1.nom.pl     | we                                                                                    |
| buddhassa                                                                        | noun | masc.dat.s   | to the Buddha, awakened one                                                           |
