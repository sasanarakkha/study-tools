| handa                                                                                          | ind  |                 | well then; now                                                             |
| mayaṃ                                                                                          | pron | 1.nom.pl        | we                                                                         |

| yo so svākkhāto bhagavatā dhammo                                                               |      |                 |                                                                            |
| yo                                                                                             | pron | masc.nom.s      | which, that … which,                                                       |
| so                                                                                             | pron | masc.nom.s      | that                                                                       |
| svākkhāto                                                                                      | adj  | masc.nom.s      | well taught, well explained                                                |
| bhagavatā                                                                                      | noun | masc.ins.s      | by Sublime One, Blessed One, fortunate one, the Buddha                     |
| dhammo                                                                                         | noun | masc.nom.s      | the Teaching, Doctrine                                                     |
| **that which well explained by the Blessed One, the Teaching,**                                |      |                 |                                                                            |
| sandiṭṭhiko akāliko ehipassiko opanayiko paccattaṃ veditabbo viññūhi                           |      |                 |                                                                            |
| sandiṭṭhiko                                                                                    | adj  | masc.nom.s      | visible, evident, presently perceivable                                    |

| opanayiko                                                                                      | adj  | masc.nom.s      | applicable, relevant, practical, useful, lit leading towards (to the goal) |
| paccattaṃ                                                                                      | adv  | ind             | for oneself, individually, personally                                      |
| veditabbo                                                                                      | ptp  | masc.nom.s      | can be known, should be understood, must be experienced                    |
