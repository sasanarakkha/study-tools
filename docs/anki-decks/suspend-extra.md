# Suspending Extra Vocabulary

If you only want to study specific chants or lessons and ignore the rest:
1. Open the **Browse** window in Anki.
2. Select your deck in the left panel.
3. Select all cards (`Ctrl + A`) and choose **Toggle Suspend** (`Ctrl + J`). Now all cards are inactive.
4. Select the specific tag for the chant/lesson you want to study.
5. Select all cards for that tag and choose **Toggle Suspend** again to reactivate them.
