# Recommended ways of studying SBS Pāḷi-English Vocab

## 1. Use the order of the Recitation Book

- Read analysis of the first chant.
- Then study words from this chant.
- After this, go to the next chant.
- Better to go with the order of the Recitation Book.

Benefits: you will cover all words consistently.

## 2. Study chants that you are interested in

Using this method you are at risk of missing some words, because they have been added in the order of the Recitation Book, and if a word appears later it is not always added again.

- Read the analysis of the chant that you like to study.
- In Anki go to **Browse**

![Anki Browse SBS Vocab deck](../../assets/images/anki-browse-sbs-vocab-deck.png)

- Select deck **SBS Pāḷi-English Vocab** in the left panel
- Select all cards with **Ctrl + A**
- Right click and choose **Toggle Suspend** (Ctrl + J)

Now all cards are inactive for study.

![Anki Browse SBS chant tags](../../assets/images/anki-browse-sbs-chant-tags.png)

- Select the chant you want to study among the tags in the left panel
- Select all cards with **Ctrl + A**
- Right click and choose **Toggle Suspend** (Ctrl + J)

![Anki Browse SBS chant selected](../../assets/images/anki-browse-sbs-chant-selected.png)

Now all cards from the selected chant will appear in your Anki daily routine. After you finish them, you may repeat the process with another chant and so on.

If English is not your first language, it is recommended to translate words into your native language. There is an empty field called "native" that you can fill in — it is preserved across updates. See [available language add-ons](../9-native.md).
