# Anki Decks

Anki is a useful tool for memorizing a large amount of information in a short period of time and for maintaining your knowledge through regular review of all the information you’ve already learned. You can read more about this tool on its [official website](https://apps.ankiweb.net/), where you can also learn how to install it.

Here’s a list of all the decks we offer. You can find most of them on AnkiWeb by [searching for “pali”](https://ankiweb.net/shared/decks?search=pali). Unfortunately, AnkiWeb frequently removes decks that don’t gain popularity.

You can find the date of the latest update for all decks on the [release GitHub page](https://github.com/sasanarakkha/study-tools/releases/latest).
