# Project Guide: SBS DhammaVinaya Learning Tools

## What it is and why

Static GitHub Pages website and GitHub Releases for SBS DhammaVinaya study materials. See https://sasanarakkha.org for details.

## Who it's for

SBS students and practitioners.

## One-off or ongoing

**Ongoing.**

## What it will produce

- MkDocs static website at `https://sasanarakkha.github.io/study-tools/`
- GitHub Releases with Anki decks and CSV exports

## How you'll know it worked

- Website builds without errors
- All navigation links work
