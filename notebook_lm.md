

# [Notebook LM](https://notebooklm.google)

It is a great AI tool that is limited only by the selected resources. It’s good for finding needed references. There are a few useful collections available:

- [Early Suttas](https://notebooklm.google.com/notebook/bccbe537-c493-4bfb-a0fc-6f3033ac79b8) - mūla of EBT*

- [Sutta Piṭaka + Commentaries](https://notebooklm.google.com/notebook/68815e7f-c5b6-4b12-a66e-9f80729b9015) - Second basket of the Tipiṭaka, mūla + atthakathā

- [Vinaya](https://notebooklm.google.com/notebook/d5bdd9b7-1b25-4e9c-b3ea-f19bb93c9081) - First basket of the Tipiṭaka, mūla part

- [Vinaya + Commentaries](https://notebooklm.google.com/notebook/118e3516-95d1-4f8d-9f8b-371ee078f621) - everything from the Tipiṭaka related to Vinaya - mūla + atthakathā + ṭīkā + aññā

- [Early Suttas + Vinaya](https://notebooklm.google.com/notebook/5f0e1b33-7b6e-4753-a1c1-1595b91ca06d) - mūla: EBT* and Vinaya

- [Pātimokkha Pali Study](https://notebooklm.google.com/notebook/36c7a255-1b16-455c-bcde-14574d8f0dd2) - sources include: Bhikkhu Vibhaṅga Pāli–English (Ven. Brahmali); Pātimokkha Analysis — normal and extended versions (Ven. Ñāṇatusita); Buddhist Monastic Code I and II (Ven. Ṭhānissaro); SBS Pātimokkha Analysis

- [Anupubbasikkhā Gradual Illumination](https://notebooklm.google.com/notebook/317a21a5-267c-4153-9543-b5bcfdcc6625?artifactId=3dcc3a5a-9b0b-488b-aa00-aa60764f6340) - everything from Tipiṭaka (mūla, atthakathā, ṭīkā) related to the Gradual Training, compiled by Ā Virāga. 

- [Pāli Tipiṭaka](https://notebooklm.google.com/notebook/bccbe537-c493-4bfb-a0fc-6f3033ac79b8) - Tipiṭaka mūla

- [Visuddhimagga](https://notebooklm.google.com/notebook/13187f85-d540-42c1-8ea8-5f723c5afab3)

---

*__EBT - Early Buddhist Texts__, it includes:
- Dīgha Nikāya, 
- Majjhimā Nikāya, 
- Saṃyutta Nikāya, 
- Aṅguttara Nikāya,
- early part of Khuddaka Nikāya: (Dhammapadapāḷi, Udānapāḷi, Itivuttakapāḷi, Suttanipātapāḷi, Theragāthāpāḷi, Therīgāthāpāḷi)

---

__mūla__ - primary text, root text

__atthakathā__ - commentaries

__ṭīkā__ - sub-commentaries

__aññā__ - this section contains many other Pāli materials
